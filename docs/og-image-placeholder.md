# OG Image Requirements

The website references `og-image.png` for social media sharing. Create a 1200x630px image with:

## Design Specs
- **Background**: Dark gradient (#0a0a0a to #0d0d0d)
- **Logo**: "FITVERSE" in large, bold, spaced letters (#10a37f green)
- **Tagline**: "we watch the algorithm so you don't have to"
- **Subtitle**: "AI-powered trend intelligence for fitness creators"
- **Visual elements**: Subtle grid pattern, fitness icons, or data visualization

## Technical Requirements
- **Size**: 1200x630px (Facebook recommended)
- **Format**: PNG or JPG
- **File size**: < 1MB
- **File name**: `og-image.png`
- **Location**: `/docs/og-image.png`

## Tools
- Canva, Figma, or Photoshop
- Use same fonts as website (system fonts)
- Ensure text is readable at small sizes (mobile preview)

## Test
After creating the image, test with:
- Facebook Sharing Debugger
- Twitter Card Validator
- LinkedIn Post Inspector

The image will appear when the site is shared on social media platforms.
